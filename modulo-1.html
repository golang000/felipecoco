Módulo 1: Introdução ao Golang

Comece sua jornada com os fundamentos da linguagem Go.

Sobre este Módulo

Neste módulo, você aprenderá os conceitos básicos de Golang, incluindo a estrutura de um programa, variáveis, tipos de dados, funções básicas e como começar a criar programas eficientes com Go.

Estrutura de um Programa Go

Um programa simples em Go começa com o pacote principal, seguido das importações necessárias e a função main():

package main

import "fmt"

func main() {
    fmt.Println("Olá, Mundo!")
}

O código acima imprime "Olá, Mundo!" na tela. Vamos detalhar cada parte:

  • package main: Define o pacote principal do programa.
  • import "fmt": Importa o pacote fmt, que fornece funções para formatar e imprimir.
  • func main(): A função principal onde a execução do programa começa.

Variáveis e Tipos de Dados

Em Go, as variáveis podem ser declaradas de várias maneiras:

var nome string = "Golang"
idade := 10
const pi = 3.14

Explicando:

  • var nome string = "Golang": Declara uma variável nome do tipo string.
  • idade := 10: Usa a atribuição curta para declarar e inicializar idade com 10.
  • const pi = 3.14: Declara uma constante pi com o valor 3.14.

Funções Básicas

Funções são blocos de código que realizam tarefas específicas. São definidas com a palavra-chave func:

func soma(a int, b int) int {
    return a + b
}

A função soma recebe dois inteiros e retorna a soma deles.

Outras Funções Importantes

Aqui estão mais exemplos de funções importantes em Go:

Função sem Retorno

func saudacao(nome string) {
    fmt.Println("Olá, " + nome + "!")
}

A função saudacao recebe um parâmetro nome e imprime uma mensagem de saudação, mas não retorna nenhum valor.

Função com Múltiplos Retornos

func dividir(a, b float64) (float64, float64) {
    quociente := a / b
    resto := a - (quociente * b)
    return quociente, resto
}

Em Go, uma função pode retornar múltiplos valores. Aqui, a função dividir retorna o quociente e o resto da divisão.

Função Anônima

anon := func() {
    fmt.Println("Esta é uma função anônima!")
}
anon()

Funções anônimas são úteis quando você precisa de uma função temporária. O exemplo acima cria e executa uma função anônima.
