Módulo 2: Conceitos Práticos em Golang

Aprofunde-se nos exemplos práticos e aprenda como usar loops, condicionais, arrays, e mais.

Sobre este Módulo

Neste módulo, você aprenderá a aplicar os conceitos básicos de Golang em exemplos práticos. Vamos explorar loops, condicionais, arrays, e como usá-los para criar programas eficientes e funcionais.

Loops em Go

O loop for é a única estrutura de loop em Go, mas ele é poderoso o suficiente para cobrir todos os casos:

for i := 0; i < 10; i++ {
    fmt.Println(i)
}

O exemplo acima imprime os números de 0 a 9.

Loop while em Go

Embora Go não tenha um loop while explícito, você pode simular um usando o loop for:

i := 0
for i < 10 {
    fmt.Println(i)
    i++
}

Esse exemplo faz o mesmo que um loop while faria em outras linguagens.

Condicionais

As estruturas condicionais em Go são similares às de outras linguagens:

if x > 10 {
    fmt.Println("x é maior que 10")
} else if x == 10 {
    fmt.Println("x é igual a 10")
} else {
    fmt.Println("x é menor que 10")
}

Você pode encadear múltiplas condições com else if e fornecer um bloco padrão com else.

Arrays e Slices

Arrays em Go têm um tamanho fixo, enquanto slices são mais flexíveis:

var arr [5]int // Array de 5 inteiros
arr[0] = 1
fmt.Println(arr)

Para usar slices, você pode fazer:

slice := []int{1, 2, 3, 4, 5}
fmt.Println(slice)

Slices são usados com mais frequência em Go devido à sua flexibilidade.
