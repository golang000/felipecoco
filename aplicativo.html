Aplicativos em Golang

Bem-vindo à seção de aplicativos em Golang! Aqui, vamos explorar um aplicativo prático de criptografia de arquivos feito em Go.

Como Funciona o Aplicativo de Criptografia

O aplicativo de criptografia permite que você proteja seus arquivos usando uma senha de 4 dígitos. Ele utiliza o algoritmo AES para criptografar e descriptografar arquivos, garantindo a segurança dos seus dados.

Funcionalidades Principais:

  • Criptografa arquivos de texto usando uma senha de 4 dígitos.
  • Descriptografa arquivos previamente criptografados usando a mesma senha.
  • Salva cópias dos arquivos originais com novos nomes, como arquivo_criptografado.txt ou arquivo_decriptografado.txt.

Explicação do Código

O aplicativo é escrito em Go e utiliza a biblioteca Fyne para a interface gráfica. Veja abaixo a estrutura básica do código:


// Exemplo de como o código funciona

import (
    "crypto/aes"
    "crypto/cipher"
    "crypto/rand"
    "io"
    "fyne.io/fyne/v2/app"
    // Importações necessárias para a interface gráfica
)

// Função principal para executar o aplicativo
func main() {
    myApp := app.New()
    myWindow := myApp.NewWindow("Aplicativo de Criptografia")
    // Configuração da interface...
}

O código utiliza o algoritmo AES para criptografia e descriptografia de dados, e a biblioteca Fyne para criar uma interface gráfica amigável. A senha é preenchida até 16 bytes para se adequar aos requisitos do AES.

Instalação e Uso

Para usar o aplicativo, siga estas etapas:

  1. Baixe o arquivo executável do aplicativo.
  2. Execute o arquivo e siga as instruções na tela.
  3. Insira uma senha de 4 dígitos para criptografar ou descriptografar seus arquivos.
