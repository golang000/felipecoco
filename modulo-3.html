Módulo 3: Algoritmos de Busca e Ordenação

Aprenda como funcionam os algoritmos fundamentais de busca e ordenação em Golang.

Sobre este Módulo

Neste módulo, exploraremos algoritmos fundamentais de busca e ordenação, como a busca linear, a busca binária, o algoritmo de ordenação por seleção, e o algoritmo de ordenação rápida (quicksort). Compreender esses algoritmos é essencial para escrever código eficiente e otimizado em Go.

Busca Linear

O algoritmo de busca linear percorre cada elemento de uma lista até encontrar o item desejado ou até que todos os elementos tenham sido verificados:

func buscaLinear(arr []int, x int) int {
    for i, v := range arr {
        if v == x {
            return i
        }
    }
    return -1 // Retorna -1 se o elemento não for encontrado
}

O exemplo acima retorna o índice do elemento x se ele for encontrado, ou -1 caso contrário.

Busca Binária

A busca binária é mais eficiente, mas requer uma lista ordenada. O algoritmo divide repetidamente a lista ao meio:

func buscaBinaria(arr []int, x int) int {
    esquerda, direita := 0, len(arr)-1
    for esquerda <= direita {
        meio := (esquerda + direita) / 2
        if arr[meio] == x {
            return meio
        } else if arr[meio] < x {
            esquerda = meio + 1
        } else {
            direita = meio - 1
        }
    }
    return -1 // Retorna -1 se o elemento não for encontrado
}

Esse algoritmo é muito mais rápido que a busca linear para listas grandes.

Ordenação por Seleção

O algoritmo de ordenação por seleção é simples, mas ineficiente para listas grandes:

func ordenacaoSelecao(arr []int) {
    n := len(arr)
    for i := 0; i < n-1; i++ {
        minIdx := i
        for j := i + 1; j < n; j++ {
            if arr[j] < arr[minIdx] {
                minIdx = j
            }
        }
        arr[i], arr[minIdx] = arr[minIdx], arr[i] // Troca os elementos
    }
}

Esse algoritmo seleciona o menor elemento e o coloca na posição correta, repetindo até que a lista esteja ordenada.

Quicksort

O algoritmo de ordenação rápida (quicksort) é muito eficiente e amplamente utilizado:

func quicksort(arr []int) []int {
    if len(arr) < 2 {
        return arr
    }
    esquerda, direita := 0, len(arr)-1
    pivot := len(arr) / 2
    arr[pivot], arr[direita] = arr[direita], arr[pivot]
    for i := range arr {
        if arr[i] < arr[direita] {
            arr[i], arr[esquerda] = arr[esquerda], arr[i]
            esquerda++
        }
    }
    arr[esquerda], arr[direita] = arr[direita], arr[esquerda]
    quicksort(arr[:esquerda])
    quicksort(arr[esquerda+1:])
    return arr
}

Quicksort é um algoritmo de divisão e conquista que ordena rapidamente grandes listas.
